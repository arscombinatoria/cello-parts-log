## 各シリーズ解説

### Chromcor
- **位置づけ**: 入門〜基礎運用の定番。
- **構造・材質（客観）**: steel strings。
- **音の傾向（主観）**: 明瞭で扱いやすい。
- **張力感・レスポンス（主観）**: 反応が速い。
- **推奨用途**: 学習用途、予備弦。
- **注意点**: 発売年は今回確認できず。

### Piranito
- **位置づけ**: 入門向け。
- **構造・材質（客観）**: steel strings。
- **音の傾向（主観）**: クリア。
- **張力感・レスポンス（主観）**: 安定重視。
- **推奨用途**: 初学者。
- **注意点**: 発売年は今回確認できず。

### Obligato
- **位置づけ**: 合成コア系を選びたい場合の中核。
- **構造・材質（客観）**: synthetic line。
- **音の傾向（主観）**: 暖かい方向。
- **張力感・レスポンス（主観）**: しなやか寄り。
- **推奨用途**: 室内楽、歌う表現。
- **注意点**: 発売年は今回確認できず。

### Evah Pirazzi / Evah Pirazzi Gold / Evah Pirazzi Soloist
- **位置づけ**: Evah 系上位群。
- **構造・材質（客観）**: いずれも cello では steel strings。
- **音の傾向（主観）**: Evah Pirazzi は明瞭でパワー寄り、Gold は密度と滑らかさ、Soloist は押し出し重視。
- **張力感・レスポンス（主観）**: いずれも高応答で、Soloist はより出力側。
- **推奨用途**: ソロ、投射力重視。
- **注意点**: 発売年は今回確認できず。

### Perpetual
- **位置づけ**: 現代的な高出力設計。
- **構造・材質（客観）**: steel strings。
- **音の傾向（主観）**: 直接的で明瞭。
- **張力感・レスポンス（主観）**: 高応答。
- **推奨用途**: 独奏、現代的セッティング。
- **注意点**: 発売年は今回確認できず。

### Permanent / Permanent Soloist
- **位置づけ**: Permanent 系。
- **構造・材質（客観）**: いずれも steel strings。
- **音の傾向（主観）**: Permanent は太さと明瞭さ、Permanent Soloist はより前に出る方向。
- **張力感・レスポンス（主観）**: いずれも速い応答で、Soloist は出力重視。
- **推奨用途**: オーケストラ〜独奏。
- **注意点**: 別シリーズとして選定する。

### Passione (Steel) / Passione (Gut)
- **位置づけ**: Passione 系。
- **構造・材質（客観）**: Steel と Gut を明確に分離したライン。
- **音の傾向（主観）**: Steel は温かさと芯、Gut は複雑な倍音。
- **張力感・レスポンス（主観）**: Gut は安定化に時間を要する場合がある。
- **推奨用途**: 音色志向に応じた選択。
- **注意点**: Gut は温湿度管理が重要。

### Oliv / Eudoxa / Chorda
- **位置づけ**: 伝統系ガットライン。
- **構造・材質（客観）**: gut strings。
- **音の傾向（主観）**: Oliv は濃密、Eudoxa は柔らかく含み、Chorda はナチュラルで軽い方向。
- **張力感・レスポンス（主観）**: 低張力〜中庸で安定化に配慮が必要。
- **推奨用途**: 古典・バロック志向。
- **注意点**: 楽器状態と環境影響を必ず確認。

### Flexocor Deluxe
- **位置づけ**: オーケストラ用途の定番候補。
- **構造・材質（客観）**: steel strings。
- **音の傾向（主観）**: 明瞭さと厚みの両立。
- **張力感・レスポンス（主観）**: 安定と反応を両立。
- **推奨用途**: オーケストラ。
- **注意点**: 発売年は今回確認できず。

### Student Cello Set
- **位置づけ**: 学習向けの組合せライン。
- **構造・材質（客観）**: Passione A/D + Chromcor G/C。
- **音の傾向（主観）**: バランス型で扱いやすい方向。
- **張力感・レスポンス（主観）**: 運用安定重視。
- **推奨用途**: 学習、初回導入。
- **注意点**: 単弦交換時は組合せバランスを再確認。
