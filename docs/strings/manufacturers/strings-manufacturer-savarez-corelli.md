## 各シリーズ解説

### Corelli Solea

- **位置づけ**: Corelli の current cello set。
- **構造・材質（客観仕様）**: handmade in France。strings wound on steel。セットは 680M / 680F。A は aluminum、D は aluminum を含む mixed metals、G/C は tungsten を含む mixed metals。
- **音の傾向（主観）**: 明瞭さと密度を両立しやすい方向。
- **張力感・レスポンス（主観）**: 反応は速め。
- **推奨用途**: 現行 Corelli cello ラインを基準にした比較。
- **注意点**: 旧来の Corelli New Concept / Corelli Gut は、今回確認できた一次資料では現行主要シリーズとして未確認。
