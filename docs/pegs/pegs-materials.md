ペグは、弦を巻き取って音程を合わせるための部品です。材質によって見た目、重量、手触り、摩耗しやすさ、湿度変化への強さが変わります。一方で、ペグ材だけによる音色差は、弦、駒、魂柱、テールピースほど大きくないため、音の変化は断定しすぎないほうが安全です。

伝統的な木製ペグでは、エボニー、ボックスウッド、ローズウッドが代表的です。Bois d’Harmonie では、フィッティング材としてボックスウッド、エボニー、ローズウッド、ペルナンブーコが挙げられており、ペグの規格情報も公開されています。Cremona Tools や Thomann の製品一覧でも、チェロ用ペグとしてエボニー、ボックスウッド、ローズウッド系の製品が多く見られます。

近年は、Wittner Finetune Pegs や Knilling Perfection Planetary Pegs のようなギア内蔵ペグも選択肢になります。これらは通常の摩擦ペグと外観を近づけつつ、内部機構によって微調整しやすくしたペグです。材質だけでなく機構も違うため、通常の木製ペグとは別分類で考えると選びやすくなります。

## 選び方の目安

見た目と伝統的な雰囲気を重視するなら、エボニー、ボックスウッド、ローズウッドから選ぶのが自然です。黒く締めたい場合はエボニー、明るく古典的に見せたい場合はボックスウッド、木目や赤みを活かしたい場合はローズウッドが候補になります。

調弦のしやすさを重視するなら、材質よりも「摩擦ペグか、ギア内蔵ペグか」の違いが大きくなります。摩擦ペグは軽く伝統的ですが、削り、ペグ穴、湿度、潤滑状態に左右されます。ギア内蔵ペグは見た目を大きく変えずに微調整しやすくできますが、取り付けには寸法確認と工房での作業が必要です。

音色への影響は、材質ごとに語られることがありますが、過度に断定しないほうがよいです。ペグはヘッド側の重量や振動条件に関係しますが、実際の音の変化は楽器個体、弦、駒、魂柱、テールピースの影響と混ざります。ページ内では「音が必ず明るくなる」「響きが必ず増える」といった断定ではなく、「重量や硬さの違いがわずかに影響する可能性がある」程度に留めます。

## チェリスト目線での判断ポイント

初心者や調弦に不安がある人は、まずペグの材質よりも、現在のペグが滑るか、固いか、微調整しにくいかを確認するとよいです。問題がある場合は、ペグコンポジションで改善することもありますが、削り直しや交換が必要な場合もあります。

中級者以上で外観も含めて楽器を整えたい場合は、ペグ、テールピース、エンドピン周辺の色味を揃えると統一感が出ます。エボニーは無難で端正、ボックスウッドはクラシック、ローズウッドは柔らかく装飾的に見えます。

練習量が多く、季節ごとの調弦ストレスを減らしたい場合は、ギア内蔵ペグも候補になります。特にスチール弦中心のチェロでは、テールピース側のアジャスターだけでなく、ペグ側で安定して細かく合わせられることが実用上の利点になります。

## 注意点

ペグ交換は、ペグ本体を買って差し替えれば終わる作業ではありません。ペグ穴のテーパー、軸径、接触面、弦穴の位置を楽器に合わせる必要があります。無理に取り付けるとペグ穴やペグボックスを傷める可能性があるため、交換やギア内蔵ペグの導入は工房に相談するのが安全です。
