# AGENTS.md

## このリポジトリの目的とスコープ
- チェロのパーツ交換・調整に関する情報を記録し、後から検証・更新できる形で共有することを目的とします。
- 主観的な所感と客観的な仕様を切り分け、経験則や参考リンクとあわせて整理します。
- 安全に関わる作業（魂柱・駒の加工など）は、専門の工房への相談を促す表現に統一します。

## コンテンツのスタイルガイド
- 主観と客観を分けて書き、条件（弦、弓、部屋の響きなど）を添えます。
- 参考文献や一次情報のリンクを可能な限り明記します。リンク切れがある場合はアーカイブの併記を検討してください。
- リスクのある作業は「推奨しない」「専門家に相談する」と記述し、断定的な手順を書かないようにします。

## AI エージェントの振る舞い
- 事実関係は一次情報を優先して確認し、確証がない場合は推測で断定せず明示します。
- 大きなファイル改変や生成を行う場合は、理由と概要を PR 説明に含めます。
- 記事の文体・粒度は既存コンテンツに合わせ、不要な書き換えや翻訳を行いません。

## コミット・PR の基本ルール
- 1 PR 1 トピックを基本とし、文章整形のみの変更は極力避け、何らかの実質的な改善を含めます。
- リンターやテストを通してから PR を作成します。CI で実行する `npm run lint`, `npm test`, `npm run build` が目安です。
- 変更内容が安全に影響する場合や大きな構成変更の場合は、README または該当ドキュメントに概要を残します。

## 作業メモ（Docusaurus 前提）
- Node.js LTS と npm を使用します。`npm ci` で依存関係を復元し、`npm start` で開発サーバーを起動できます。
- Docs は `/docs` 配下に配置し、Front Matter を用いてタイトルやスラッグを設定します。サイトのルートは `docs/intro.md` を参照してください。
