## 各シリーズ解説

### Alphayue
- **位置づけ**: 入門ライン。
- **構造・材質（客観）**: A/D = carbon steel、G/C = rope core。
- **音の傾向（主観）**: 素直で扱いやすい方向。
- **張力感・レスポンス（主観）**: 安定重視。
- **推奨用途**: 初学者、学校用途。
- **注意点**: 発売年は今回確認できず。

### Dominant
- **位置づけ**: 合成コア系の基準。
- **構造・材質（客観）**: synthetic core。
- **音の傾向（主観）**: 暖かめで中庸。
- **張力感・レスポンス（主観）**: 中庸。
- **推奨用途**: 学習〜アンサンブル。
- **注意点**: 発売年は今回確認できず。

### Spirocore
- **位置づけ**: 代表的定番ライン。
- **構造・材質（客観）**: spiral core。巻線バリエーションがある。
- **音の傾向（主観）**: 明瞭で芯が強い。
- **張力感・レスポンス（主観）**: 速い反応。
- **推奨用途**: オーケストラ、ソロ、低弦強化。
- **注意点**: 巻線違いで印象差が大きい。

### Belcanto Gold
- **位置づけ**: 上級の歌いやすさ重視ライン。
- **構造・材質（客観）**: steel 系。
- **音の傾向（主観）**: 深さと滑らかさを両立しやすい方向。
- **張力感・レスポンス（主観）**: しなやか寄り。
- **推奨用途**: 室内楽、歌う表現。
- **注意点**: 現行名称は Belcanto Gold。

### Versum / Versum Solo
- **位置づけ**: 現代的な上位ライン。
- **構造・材質（客観）**: A/D = carbon steel + multialloy、G/C = spiral core + tungsten/chrome。
- **音の傾向（主観）**: Versum は密度と明瞭さ、Versum Solo はより前に出る方向。
- **張力感・レスポンス（主観）**: いずれも高応答で、Solo は出力重視。
- **推奨用途**: ソロ、現代的セッティング。
- **注意点**: 発売年は今回確認できず。

### Rondo
- **位置づけ**: current official の advanced/pro line。
- **構造・材質（客観）**: 上位 steel 系設計。Rondo Experience A string の展開あり。
- **音の傾向（主観）**: 力強い音像を狙いやすい。
- **張力感・レスポンス（主観）**: 反応が速い。
- **推奨用途**: 独奏、投射力重視。
- **注意点**: 発売年は今回確認できず。
